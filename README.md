# go-simple-blog

## Requirements

 - go 1.13
 - MySQL

## How to use

### Clone

```bash
$ git clone git@github.com:k725/go-simple-blog.git
$ cd go-simple-blog
```

### Install depends

```bash
$ go mod download
```

### Build

```bash
$ go build . 
```

### Run

NOTE: MySQL connection information can be found in the startup script

for Windows

```bat
> run.bat
```
for Darwin/Linux

```bash
$ ./run.sh
```
